Software Used
  • Adobe Photoshop
  • Adobe Illustrator

Kongregate Publishing would send one-page sell sheets to each platform whenever we had a major game update or event. The sell sheet would include game art and info about the update or event.

As the designer, I would be provided with a logo, basic copy, artwork and game assets from which to create the design. Often I would work closely with the game’s producer on rewriting copy requesting additional assets in order to create the final product.
