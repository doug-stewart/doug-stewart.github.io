When Kongregate Publishing released a new game, or a game had a major update, we would promote it across Kongregate.com with home page features, site skins, on site-ads and a landing page to direct new users to the appropriate gaming platform.

To create these assets I would be provided with the required logos, fonts, copy, key art and any relevant game assets. These would be modified and composited as necessary to create the final result.
